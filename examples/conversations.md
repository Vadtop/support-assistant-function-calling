# Примеры диалогов с Support Assistant

## Диалог 1: Вопрос про баланс (с использованием tool)

**User:** Как узнать баланс по карте?

**Assistant:** Вы можете узнать баланс по карте следующими способами:

1. В мобильном приложении банка в разделе «Карты».
2. Позвонив по телефону службы поддержки: 8-800-123-45-67.

*Tool used: search_faq*  
*Tool query: "как узнать баланс по карте"*

---

## Диалог 2: Общий вопрос (без tool)

**User:** Привет!

**Assistant:** Здравствуйте! Как я могу помочь вам сегодня?

*Tool used: none*

---

## Диалог 3: Вопрос про лимиты (с использованием tool)

**User:** Хочу увеличить лимит на снятие наличных

**Assistant:** Чтобы увеличить лимит на снятие наличных, вы можете сделать это через мобильное приложение вашего банка. Перейдите в раздел "Карты", затем "Настройки" и выберите "Лимиты". Максимальный лимит на снятие наличных составляет 200 000 ₽ в сутки.

*Tool used: search_faq*  
*Tool query: "увеличить лимит на снятие наличных"*

---

## Архитектура диалога с tool

User Question → LLM (system prompt + tools definition)
↓
[Decision: use tool?]
↓
YES: search_faq(query)
↓
FAQ results → LLM
↓
Assistant Answer (based on FAQ)
